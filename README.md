## 安裝

前往[Node.js官網](https://nodejs.org/)下載並安裝最新版本的Node.js和npm。

環境需求設定：

```bash
git clone https://github.com/AnnYang81/air_map.git
cd air_map
npm init -y
npm install express axios swagger-jsdoc swagger-ui-express
yarn add dotenv
```


啟動伺服器：
```bash
node route.js
```

測試API：
```bash
http://localhost:3000/api-docs
```
